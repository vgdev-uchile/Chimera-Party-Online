Name:
  Rodents (Rat Rework)
Version:
  1.0
Description:
  Rodent sprites reworked from Reemax's rat. Created as a drop-in
  replacement for Stendhal.
Sprite details:
  - Orientation: orthogonal (N/E/S/W)
  - Dimensions: 32x32
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
